# Volt Eligibility
This is a simple project to check if you are eligible for the Volt program. It will ask your mobile number and PAN card number, and based on that information, then tell you whether or not you qualify.

## Framework 
The framework used here is Pytest which is a micro web framework written in Python. 

## Setup and run instructions
1. Clone this repository.
2. Install all dependencies using pip install -r requirements.txt
3. Run pytest test.py
